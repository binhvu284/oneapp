# OneApp Data

This folder contains all database schemas and data structures for OneApp Database.

## Structure

- **Schemas**: Database schema definitions for OneApp Database tables
- **Data**: Sample data and migrations for OneApp Database

## Database Tables

### In use App
Stores information about all in-use applications in the OneApp system.

### Categories
Stores category information for organizing applications and content.

## Notes

- This is the OneApp Database (local storage)
- Data stored here cannot be restored if the project is deleted
- All schemas are defined in the OneApp Data page UI
